 2020/07/10

安装

常用命令

scoop bucket add xxx
scoop install xxxx
scoop uninstall xxxx
scoop search xxxx
scoop update * 更新所有已安装软件
scoop install xxxx@version
scoop reset xxxx 切换软件版本
scoop info xxxx
