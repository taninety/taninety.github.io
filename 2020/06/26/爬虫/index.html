项目介绍

该项目通过抓取 Twitter 及 Instagram HTTP Response 返回的 json 文本来获取网页内容。不需要使用 API。灵感来源于这篇文章。

可以实现的功能包括

  • 输入一组 Twitter / Instagram条目链接，获得包含正文全部内容和当前用户信息的 json 文件，以及条目中涉及的图片和视频文件

使用步骤

下载需要的软件

  • browserup-proxy
  • chromedriver

安装 browserup-proxy-py 包

  • pip install git+https://github.com/browserup/browserup-proxy-py.git@master#egg=browserup-proxy==0.1.0

设置脚本中的常量

  • proxies: python requests 命令使用的代理
  • browserupproxy_path: browserup-proxy 可执行文件存放的路径 r'*******\browserup-proxy.bat'
  • chromedriver_path: chromedriver 可执行文件存放的路径 r'*******\chromedriver.exe'
  • instagram_username: instagram爬虫登录用的用户名
  • instagram_password: instagram爬虫登录用的密码
  • instagram_cookie_file: instagram临时cookies文件存放的位置，无需修改

创建输入文件

  • 文件名标准格式为 yyyymm.urls
  • 每行放置一条 twitter 或 instagram 网址，以 https 开头
  • 支持两种注释，开头前四个字母不是 http 的行被视为注释，每行空格之后的内容被视为注释

配置合适的代理规则

  • java.exe 配置Proxifier代理，确保 browserup-proxy 能够连接 Twitter 和 Instagram服务器
  • chrome.exe 及 python.exe 不配置Proxifier代理，否则会影响 browserup-proxy 运行
  • python requests 方法 设置http及https代理，详见设置脚本中的常量
