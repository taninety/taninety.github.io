//暂时放弃红黑树的插入删除操作，太复杂了

五条性质

  1. 树的所有节点，要么是红色，要么是黑色
  2. 树的根节点是黑色
  3. 树中的叶子节点是黑色
  4. 红色节点的子节点一定是黑色，也就是没有两个联在一起的红色节点
  5. 每个节点到任意一个叶子节点，经过的黑色节点数量是相等的

两个推论

  1. 红黑树从根节点到叶子节点的最长路径，不会大于最短路径的两倍
  2. 一棵含有n个节点的红黑树的高度之多为2log(n+1)

最小堆

标准库实现 java.util.PriorityQueue

//用数组实现的完全二叉树
public class MyMinHeap {
    private int size;
    private int capacity;
    private int[] heap;

    public MinHeap(int initialCapacity) {
        this.size = size;
        this.capacity = initialCapacity;
        this.heap = new int[initialCapacity];
    }

    //向树中插入元素 O(logn)
    public void add(int value) { heapifyUp() }

    //获取树顶元素(最小值) O(logn)
    public int poll() { heapifyDown() }

    //向上调整 O(logn)
    private void heapifyUp()

    //向上调整 O(logn)
    private void heaapifyDown()
}

散列表 (Hash Table)

标准库实现 java.util.HashMap

class Entry {
    int key;
    int value;

    public Entry(int key, int value) {
        this.key = key;
        this.value = value;
    }
}
public class MyHashMap {
    private static final int DEFAULT_CAPACITY = 16;
    private static final double LOAD_FACTOR = 0.75;

    private LinkedList<Entry>[] table;
    private int size;

    public MyHashMap() {
        table = new LinkedList[DEFAULT_CAPACITY];
        size = 0;
    }

    //插入元素 O(1)
    public void put(int key, int value) { resize(); }

    //获取元素 O(1)
    public int get(int key)

    //移除元素 O(1)
    public void remove(int key) 

    //获取容量 O(1)
    public int size()

    //扩容 O(1)?
    private void resize()

    //哈希值 O(1)
    private int hash(int key)
}
  • HashMap使用数组和链表实现，综合了数据和链表的优点
  • HashSet的底层实现原理也是HashMap，只不过value值存了一个相同的静态对象

class Vertex {
